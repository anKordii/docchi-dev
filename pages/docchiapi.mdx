# Docchi API Reference

| Products                        | Description |
| ------------------------------- | ----------- |
| [Series](/docchiapi/series)     | Gets information about one or more series. |
| [Episodes](/docchiapi/episodes) | Gets the episodes list editors. |
| [Profiles](/docchiapi/profiles) | Gets information about one or more profiles. |
| [Comments](/docchiapi/comments) | Gets comments for one or more specified comments sets. |
| [Reactions](/docchiapi/reactions) | Gets reactions for one or more specified reactions sets. |
| [Hentai](/docchiapi/hentai) | Gets information about one or more hentai. |
