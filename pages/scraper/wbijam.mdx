# Wbijam

---

### Basic Usage

```javascript
import Scraper from "@docchi/scraping-anime-websites-poland";

console.log(
  await Scraper({
    anime: "narutoboruto",
    episode: "boruto-275",
    website: "wbijam",
  })
);
```

### Parameters

| Parameter | Type               | Example      | Information                                              |
| --------- | ------------------ | ------------ | -------------------------------------------------------- |
| `anime`   | string             | narutoboruto | Use subdomain e.g. `https://` narutoboruto `.wbijam.pl/` |
| `episode` | string | boruto-275   |
| `website` | string             | wbijam       |
