# Desu-Online

---

### Basic Usage

```javascript
import Scraper from "@docchi/scraping-anime-websites-poland";

console.log(
  await Scraper({
    anime: "koukyuu-no-karasu",
    episode: 5,
    website: "desuonline",
  })
);
```

### Parameters

| Parameter | Type               | Example           |
| --------- | ------------------ | ----------------- |
| `anime`   | string             | koukyuu-no-karasu |
| `episode` | integer `/` string | 5                 |
| `website` | string             | desuonline        |

<details>
    <summary>Responses</summary>

#### Response `200`

```json
{
  "status": 200,
  "message": "Success",
  "episode_url": [
    { "player": "CDA", "url": "https://ebd.cda.pl/620x395/1340344027" }
  ],
  "episode_next_url": 6
}
```

#### Response `500`

```json
{ "status": 500, "message": "Something went wrong!" }
```

</details>
