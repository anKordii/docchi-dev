# Reiko Project

---

### Basic Usage

```javascript
import Scraper from "@docchi/scraping-anime-websites-poland";

console.log(
  await Scraper({
    anime: "p/liang-bu-yi",
    episode: 6,
    website: "reikoproject",
  })
);
```

### Parameters

| Parameter | Type               | Example       |
| --------- | ------------------ | ------------- |
| `anime`   | string             | p/liang-bu-yi |
| `episode` | integer `/` string | 6             |
| `website` | string             | reikoproject  |
