# Paldea

---

### Basic Usage

```javascript
import Scraper from "@docchi/scraping-anime-websites-poland";

console.log(
  await Scraper({
    episode: "hz001",
    website: "paldea",
  })
);
```

### Parameters

| Parameter | Type   | Example |
| --------- | ------ | ------- |
| `episode` | string | hz001   |
| `website` | string | paldea  |
