# Bitly

---

### Basic Usage

```javascript
import Scraper from "@docchi/scraping-anime-websites-poland";

console.log(
  await Scraper({
    url: "https://bit.ly/masamunes2rcda2",
    website: "bitly",
  })
);
```

### Parameters

| Parameter | Type   | Example                        |
| --------- | ------ | ------------------------------ |
| `url`     | string | https://bit.ly/masamunes2rcda2 |
| `website` | string | bitly                          |
