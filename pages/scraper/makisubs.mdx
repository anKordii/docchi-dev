# MakiSubs

---

### Basic Usage

```javascript
import Scraper from "@docchi/scraping-anime-websites-poland";

console.log(
  await Scraper({
    anime: "jitsu-wa-ore-saikyou-deshita-1",
    episode: 1,
    website: "maki",
  })
);
```

### Parameters

| Parameter | Type               | Example                        |
| --------- | ------------------ | ------------------------------ |
| `anime`   | string             | jitsu-wa-ore-saikyou-deshita-1 |
| `episode` | integer `/` string | 1                              |
| `website` | string             | maki                           |
