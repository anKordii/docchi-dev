# NanaSubs

---

### Basic Usage

```javascript
import Scraper from "@docchi/scraping-anime-websites-poland";

console.log(
  await Scraper({
    anime: "vinland-saga-season-2",
    episode: 9,
    website: "nanasubs",
  })
);
```

### Parameters

| Parameter | Type               | Example               |
| --------- | ------------------ | --------------------- |
| `anime`   | string             | vinland-saga-season-2 |
| `episode` | integer `/` string | 9                     |
| `website` | string             | nanasubs              |
