# Orfeusz Subs

---

### Basic Usage

```javascript
import Scraper from "@docchi/scraping-anime-websites-poland";

console.log(
  await Scraper({
    anime: 10,
    episode: 1,
    website: "orfeusz",
  })
);
```

### Parameters

| Parameter | Type               | Example |
| --------- | ------------------ | ------- |
| `anime`   | integer `/` string | 10      |
| `episode` | integer `/` string | 1       |
| `website` | string             | orfeusz |

<details>
    <summary>Responses</summary>

#### Response `200`

```json
{
  "status": 200,
  "message": "Success",
  "episode_url": [
    { "player": "cda", "url": "https://ebd.cda.pl/620x368/1510259904" }
  ],
  "episode_next_url": 2
}
```

#### Response `500`

```json
{ "status": 500, "message": "Something went wrong!" }
```
