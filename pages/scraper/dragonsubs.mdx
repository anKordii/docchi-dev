# DraGoN-Subs

---

### Basic Usage

```javascript
import Scraper from "@docchi/scraping-anime-websites-poland";

console.log(
  await Scraper({
    anime: "saikyou-onmyouji-no-isekai-tenseiki",
    episode: 1,
    website: "dragon",
  })
);
```

### Parameters

| Parameter | Type               | Example                         |
| --------- | ------------------ | ------------------------------- |
| `anime`   | string             | saikyou-onmyouji-no-isekai-tenseiki |
| `episode` | integer `/` string | 1                               |
| `website` | string             | dragon                          |
