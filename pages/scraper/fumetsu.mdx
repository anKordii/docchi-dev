# Fumetsu

---

### Basic Usage

```javascript
import Scraper from "@docchi/scraping-anime-websites-poland";

console.log(
  await Scraper({
    anime: "TenseiShitaraKenDeshita",
    episode: 1,
    website: "fumetsu",
  })
);
```

### Parameters

| Parameter | Type               | Example                 |
| --------- | ------------------ | ----------------------- |
| `anime`   | string             | TenseiShitaraKenDeshita |
| `episode` | integer `/` string | 1                       |
| `website` | string             | fumetsu                 |
