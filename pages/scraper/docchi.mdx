# Docchi

---

### Basic Usage

```javascript
import Scraper from "@docchi/scraping-anime-websites-poland";

console.log(
  await Scraper({
    anime: "nanatsu-no-maken-ga-shihai-suru",
    episode: "1",
    website: "docchi",
  })
);
```

### Parameters

| Parameter | Type               | Example                         |
| --------- | ------------------ | ------------------------------- |
| `anime`   | string             | nanatsu-no-maken-ga-shihai-suru |
| `episode` | integer `/` string | 1                               |
| `website` | string             | docchi                          |
