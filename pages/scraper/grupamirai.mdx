# Grupa-Mirai

---

### Basic Usage

```javascript
import Scraper from "@docchi/scraping-anime-websites-poland";

console.log(
  await Scraper({
    anime: "blue",
    episode: 3,
    website: "grupamirai",
  })
);
```

### Parameters

| Parameter | Type               | Example    |
| --------- | ------------------ | ---------- |
| `anime`   | string             | blue       |
| `episode` | integer `/` string | 3          |
| `website` | string             | grupamirai |
