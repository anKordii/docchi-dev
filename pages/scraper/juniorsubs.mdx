# Junior-Subs

---

### Basic Usage

```javascript
import Scraper from "@docchi/scraping-anime-websites-poland";

console.log(
  await Scraper({
    folder: "hentai",
    anime: "yuutousei-ayaka-no-uraomote",
    episode: "01",
    website: "junior",
  })
);
```

### Parameters

| Parameter | Type   | Example                     |
| --------- | ------ | --------------------------- |
| `folder`  | string | hentai                      |
| `anime`   | string | yuutousei-ayaka-no-uraomote |
| `episode` | string | 01                          |
| `website` | string | junior                      |
