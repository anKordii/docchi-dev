# FrixySubs

---

### Basic Usage

```javascript
import Scraper from "@docchi/scraping-anime-websites-poland";

console.log(
  await Scraper({
    anime: "spy-x-family",
    episode: "1",
    website: "frixysubs",
  })
);
```

### Parameters

| Parameter | Type               | Example      |
| --------- | ------------------ | ------------ |
| `anime`   | string             | spy-x-family |
| `episode` | integer `/` string | 10           |
| `website` | string             | frixysubs    |
