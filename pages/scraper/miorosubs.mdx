# Mioro-Subs

---

### Basic Usage

```javascript
import Scraper from "@docchi/scraping-anime-websites-poland";

console.log(
  await Scraper({
    anime: "blue-lock-1",
    episode: 22,
    website: "miorosubs",
  })
);
```

### Parameters

| Parameter | Type               | Example     |
| --------- | ------------------ | ----------- |
| `anime`   | string             | blue-lock-1 |
| `episode` | integer `/` string | 22          |
| `website` | string             | miorosubs   |
