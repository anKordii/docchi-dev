# CDA Folder

---

### Basic Usage

```javascript
import Scraper from "@docchi/scraping-anime-websites-poland";

console.log(
  await Scraper({
    user: "SziszoweSubki",
    folder: 43344342,
    type: "spaces",
    spaces: 5,
    episode: 1,
    website: "cda",
  })
);
```

### Parameters

| Parameter | Type             | Example  | Information                                                                                                                                            |
| --------- | ---------------- | -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `user`    | string           | docchi   | ![example](/img/link.png)                                                                                                                              |
| `folder`  | string`/`integer | 37956235 |                                                                                                                                                        |
| `type`    | string           | spaces   | You have two types to choose from, one is `spaces` and the other one is `s0e0` which you should use when episode number is presented in S01E01 format. |
| `spaces`  | string`/`integer | 3        | ![example](/img/spaces.png) If you are using type `s0e0` you also need to determine where the S01E01 format is located.                                |
| `episode` | string`/`integer | 1        | You have two options to choose from one is to type number e.g. `1, 2 or 12`, or you can just type `all` and you will receive all episodes from folder. |
| `website` | string           | cda      |                                                                                                                                                        |

#### Response `204`

Episode is missing.

```json
{
  "status": 204,
  "message": "No Content!"
}
```

#### Response `400`

Wrong type of request.

```json
{
  "status": 400,
  "message": "Unknown type of request. Acceptable (spaces, s0e0)"
}
```
