# Okami-Subs

---

### Basic Usage

```javascript
import Scraper from "@docchi/scraping-anime-websites-poland";

console.log(
  await Scraper({
    anime: "arknights-animation-prelude-to-dawn",
    episode: "3-c77ffb6d-275d-4422-a3f9-a68162dd9a5d",
    website: "okamisubs",
  })
);
```

### Parameters

| Parameter | Type   | Example                                |
| --------- | ------ | -------------------------------------- |
| `anime`   | string | arknights-animation-prelude-to-dawn    |
| `episode` | string | 3-c77ffb6d-275d-4422-a3f9-a68162dd9a5d |
| `website` | string | okamisubs                              |
