# Reactions Endpoints

---

### Gets reactions for specific location

`GET` ‚ùØ `https://api.docchi.pl/v1/reactions/find/{LOCATION}?type={LOCATION_TYPE}`

### Parameters

| Parameter | Type           | Example |
| --------- | -------------- | ------- |
| `LOCATION`      | string/integer | 2185    |
| `LOCATION_TYPE` | string | `comment`/`episode`/`community` |

```json
[{
    "name": "like",
    "count": 1
}]
```

```json

    "message": "Not found reactions ( 2185 )."
}
```
