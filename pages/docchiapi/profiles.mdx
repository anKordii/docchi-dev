### Parameters

| Parameter | Type           | Example |
| --------- | -------------- | ------- |
| `XID`     | string/integer | 45      |

<details>
    <summary>Responses</summary>

#### Response `200`

```json
{
  "episodes": 1817,
  "series": 229
}
```

```json
{
  "episodes": null,
  "series": 0
}
```
